# My Next Step

Your AI coach that creates clear, actionable steps and makes them easy to do.

## Setup

1. Clone this repo
2. Run `npm install`
3. Run `npm run dev` to start locally
4. Deploy to Vercel by connecting this repo

## How it works

- **Steps tab**: Actionable to-dos with direct links to do them
- **Plans tab**: Multi-step goals broken into trackable tasks
- **Coach tab**: Chat with AI to build your steps and plans
- **Feedback loop**: Mark steps done, tell the AI how it went, get better recommendations

Built with React + Vite. AI powered by Claude API.
